# Desafío Técnico WEB

## Descripción

Proyecto desarrollado con HTML, CSS y JavaScript.

La aplicación web contiene dos simuladores interactivos:

1. Simulador de Transferencia de Calor
2. Calculador de Combinaciones Complejas

---

## Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript

---

## Funcionalidades

### Ejercicio 1: Transferencia de Calor

Calcula la temperatura final de un objeto usando la fórmula:

T = Ts + (T0 - Ts) * e^(-k * t)

El sistema utiliza:

* Math.exp()
* Math.round()

---

### Ejercicio 2: Combinaciones

Calcula el total de combinaciones posibles utilizando:

C(n,r) = n! / (r!(n-r)!)

El programa implementa:

* Función factorial propia
* Validaciones de entrada
* Cálculo dinámico de resultados
